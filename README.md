# epg
央视，各地方、港澳台 最近三天节目表 每日更新！
使用TVXML格式，只要支持tvxml格式的播放器即可使用，像perfect player,tivimate等  
提供xml及xml.gz， xz暂不提供（因在播放器支持不友好）  
[点击此处](http://epg.51zmt.top:8000 "提供节目表频道列表")查看提供节目表的频道列表！  
也可以对频道进行自动匹配  
请保持tvg-name与列表中相同
